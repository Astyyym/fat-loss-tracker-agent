# 公开仓库通用化改造计划

**目标：** 移除公开文档中的本地绝对路径和固定周期设定，让计划周期和营养目标可配置。

### 已实施

- README 与 prompt 改成相对路径示例，不包含个人电脑目录。
- 新增 `config.example.json`；用户复制为本地 `config.json` 后可设置 `program_weeks` 和营养目标。
- `config.json`、健康数据、摘要、卡片和备份均由 Git 忽略。
- 后端、今日卡片、仪表盘和趋势参考线读取同一份配置。
- 项目名称调整为“减脂记录 Agent”。
- GitHub 仓库同步改为不带固定周期的名称。
